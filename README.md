# 🎨 TomoTexture

A tool to replace and manage UGC canvases in your Tomodachi Life save files — swap any drawing with your own image!

> 🚨 **Only food textures are supported for now.**

---

## 🚀 Getting Started

1. **Download** `TomoTexture.exe` from the [latest release](https://github.com/AlfonsoMallozzi/TomoTexture/releases).
2. **Run** the app — no installation needed.
3. If the app doesn't find your save folder automatically (it should if you use Ryujinx Canary with its folder in AppData), click **"Browse"** to select it manually.
4. Pick a canvas, click **Replace**, choose your image, and you're done!

---

## ⚠️ Heads Up

- This is a **beta** — back up your save folder before using it.
- Make sure your emulator/game is **closed** before replacing textures.
- Found a bug? [Open an issue!](https://github.com/AlfonsoMallozzi/TomoTexture/issues)
