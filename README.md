# 🎨 TomoTexture


<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/ec1270e9-c062-4360-9f83-64eb5c232a5e" />

A tool to replace and manage UGC canvases in your Tomodachi Life save files — swap any drawing with your own image.

## Currently Supported

- Foods  
- Goods (except videos)  
- Faces  

## How to Use

### 1. Select Save Folder
Select the folder containing your save files (the game slots).

<img width="1100" height="825" alt="image" src="https://github.com/user-attachments/assets/af878ba7-ac70-4aa1-9565-c32978d63a54" />

You will be prompted to choose a backup folder. Select where you want backups to be stored.

---

### 2. Choose and Replace
Select the drawing you want to swap and choose how the image will be fitted to the in-game sprite.  
(If not prompted, you will be taken to a confirmation screen.)

<img width="753" height="552" alt="image" src="https://github.com/user-attachments/assets/dae8e5fb-163f-4a90-b0a3-0d7f3edfcc5e" />

---

### 3. Done
Your new sprite should now appear in-game.

<img width="1099" height="826" alt="image" src="https://github.com/user-attachments/assets/1a6ce3eb-8002-4567-92fc-e864c9c0ef7a" />

You can revert changes if needed.
